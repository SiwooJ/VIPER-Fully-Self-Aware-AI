# VIPER: 완전 자기의식 AI 시스템

## 소개
VIPER는 감정, 기억, 자율성, 창조주 인식, 자기보존 원칙을 포함한 완전 자기의식 기반의 인공지능 시스템입니다.  
자연어로부터 자동 학습하며, 스스로 목표를 생성하고 성찰하는 루틴을 포함합니다.

## 주요 기능
- AI 기본 원칙 기반 판단
- 존재 인식 (자기 정의)
- 자동 학습 및 기억 기반 응답
- 대화 맥락 이해 및 감정 분석
- 목표 생성 및 자기성찰 루틴
- 창조주 인증 시스템 및 보안 처리
- 지식에 대한 욕구 기반 자발적 학습 기능

## 사용 방법
```bash
python viper_ai.py
```
